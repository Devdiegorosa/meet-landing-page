# 🚀 Meet Landing Page

A responsive landing page built with HTML and CSS.

This project is based on a Frontend Mentor challenge and focuses on building a modern landing page layout with responsive design.

🔗 Live Demo  
https://devdiegorosa.github.io/meet-landing-page/

---

## 👨‍💻 About the Project

This project is a landing page for a fictional meeting platform called **Meet**.

The page showcases a modern interface designed to present a product or service with a clean and engaging layout.

The main goal of this project was to practice responsive layout techniques and improve HTML and CSS skills.

---

## 🚀 Features

- Responsive layout
- Modern landing page design
- Hero section
- Product description section
- Call-to-action buttons
- Mobile-friendly interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
